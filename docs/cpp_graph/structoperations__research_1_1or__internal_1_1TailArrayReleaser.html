template<typename GraphType, bool has_reverse_arcs>
struct operations_research::or_internal::TailArrayReleaser< GraphType, has_reverse_arcs >

The TailArrayReleaser class template is not expected to be used by clients.

It is a helper for the TailArrayManager template.

The TailArrayReleaser for graphs with reverse arcs does nothing.
